# Audio Synthesizer

### Features
- Real Time Audio Synthesis

### Hardware
- STM32 L432KC
- Simple Buttons and Wiring
- Speaker
- PAM8302A Amplifier

### Software
- DMA-driven output to DAC using buffer with half and complete transfer interrupts
- GPIO EXTI for input
- Timer-triggered sampling
- Additive Synthesis for chords

- In Progress
